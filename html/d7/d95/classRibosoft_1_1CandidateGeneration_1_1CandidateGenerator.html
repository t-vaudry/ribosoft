The core of the program, the main functionality. This function builds the tree structure of nodes to generate the candidates set up for evaluation to then be ranked. Here is a summary of the process. A- Get all permutations of ribozyme sequence: a) Generate structure b) Do regular traversal BUT if current idx is in list of RNA link indices, just add X and continue B- Generate cut site tree C- Traverse cut site tree to generate a list of all possible cut sites D- Foreach cut site, if not found in RNA input, delete. E- Foreach remaining cutsite: a) Find complement i) foreach ribozyme sequence (A-), copy, and replace Xi with list of RNA link indices[i] (ignoring any -). Add to list to send to algo F- Foreach repeat notation: a) Add to existing valid cutsites b) Find which of these new cutsites are valid c) Complete new ribozyme with this new cutsite G- Send list generated in E- to algo.
